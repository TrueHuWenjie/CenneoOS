## 线程、进程及任务 ##


| 名称 |   运行空间   |              虚拟内存空间              |
| ---- | ------------ | -------------------------------------- |
| 任务 |  只在内核态  |            共享的内核态空间            |
| 线程 | 主要在用户态 | 与当前进程下的所有线程共享虚拟内存空间 |
| 进程 | 主要在用户态 |            有自己的虚拟内存            |

## 线程状态 ##

| 状态 |       描述       |
| ---- | ---------------- |
| 实时 |  最高执行优先级  |
| 睡眠 | 一段时间内不运行 |
| 僵尸 |    等待父进程    |
